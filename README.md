Исходные данные:
- Имеется состав документов (чертежи деталей, сборочные чертежи и спецификации) с незаполненной основной надписью (фамилии, даты, литера). 
- В основных надписях неверные данные, либо изменилась ФИО подписанта, дата или литера.

Цель:
- Сэкономить время ручного перебивания основной надписи для большого количества документов.
- Получить комплект документов с одинаково отредактированными данными основных надписей.

Порядок работы:
- Выбрать необходимые файлы для внесения изменений.
- Заполнить поля "ФИО" и "дата".
- При необходимости написать литеру.
- При необходимости можно подготовить PDF.
- Нажать "Начать".
- Ожидать сообщения об удачном завершении.

Примечание:
- Необходимо заносить все данные. Если поле утилиты будет пустым, то и в основной надписи это поле тоже будет пустым.
- Компас должен быть запущен заранее.
